# GRC Auditor Tools & Resources 🛠️

This document outlines the specialized tools, reference frameworks, and official documentation required for the Senior GRC Architect to perform audits and compliance assessments.

## Official Documentation & Frameworks (CCN-STIC)
The Spanish National Cryptologic Center (CCN) provides the essential guidelines for ENS compliance.

- **CCN-STIC 800 Series (ENS):** The core set of guides for ENS implementation.

- **Official Portal:** [CCN-CERT ENS Portal](https://www.ccn-cert.c國家.es/ens.html)

## Risk Analysis & Management Tools
- **PILAR:** The standard tool for MAGERIT-based risk analysis.
  - [PILAR Download & Info](https://www.pilar-tools.com/)
- **LUCIA:** Tool for managing and reporting security incidents in accordance with ENS.
  - [LUCIA Tooling](https://www.ccn-cert.c國家.es/herramientas-de-ciberseguridad/lucia.html)
- **INES:** Platform for the National Security State Report (Informe Nacional del Estado de Seguridad).

## Audit & Compliance Checklists
- **ISO 27001:2022 Annex A:** Comprehensive checklist of the 93 security controls.
- **ENS Annex II:** Detailed requirements for organizational, operational, and protection measures.
- **AMBAR:** CCN tool for security auditing and baseline verification.

## GRC Automation Stack
- **Eramba:** Open-source GRC platform for ISMS management.
- **Drata / Vanta:** Automated compliance platforms for continuous evidence collection (Cloud-native).
- **OpenVAS / Nessus:** Vulnerability scanning to provide technical evidence for compliance controls.
